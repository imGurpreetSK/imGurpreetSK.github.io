Some events close to my heart

Magicbricks hackathon - Winners
Created a chatbot app that takes user input as a string and processes it to fetch location details. After this, the information is used to show related properties to user along with images, address, price and owner details as provided by MagicBricks API. We also proposed adding 360 VR imagery for good user experience and provided a demo for the same.

HackDTU - Synergy'17
We created an accessible health-care app using OpenCV and android to calculate user heart-rate from real-time forehead video and heart-beat readings, along with ability to find nearest hospitals, health-care centers, trauma centers, etc on map and contacts for all these authorities and other registered individuals which can be contacted in event of need.

Smarter than yesterday hackathon
Created a 3D game for children using Unity3D and C# scripting and was one of the 15 selected teams (from over 4200 registrants) called for the final demo held in Bangalore.

AngelHack Jaipur - Runner-Ups
Created a text summarization portal in 20 hours at world's biggest hackathon series where user can input the text or scan a document and can get the summary of the same for sharing or storage purposes. User can also read-aloud the text and translate the same to some other language.

IncubateX-IND - Runner-ups
IncubateIND is an entrepreneurship hackathon, where participants with diverse skillsets collaborate to create awesome products. We created an AR solution for C-class executives to effectively share media and information among themselves and save time by converting text to video.
